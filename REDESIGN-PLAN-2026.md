### 2. School ERP System (Comprehensive Educational Management)

| Problem | Impact |
|---------|--------|
| Schools using paper-based records | Data loss, retrieval delays |
| Multiple disconnected software tools | Staff confusion, data inconsistency |
| Manual attendance tracking | Time-consuming, error-prone |
| No centralized reporting | Difficult to track student performance |
| Excel dependency | Version conflicts, no real-time collaboration |

**Stakeholder Pain Points:**
- **Administrators:** Needed unified view of all operations
- **Teachers:** Wanted quick attendance and grade entry
- **Parents:** Needed visibility into child's performance
- **HR:** Manual payroll and leave tracking

### 3. Mental Health Application for Young Athletes

#### CASE STUDY

**Category:** `Mobile` `Health & Wellness` `React Native` `AI/Personalization`
**Duration:** 4+ months
**Role:** Complete System Design, Feature Architecture, Development (Solo)

---

**📋 OVERVIEW**

I designed and developed a mental health and performance improvement application specifically for young athletes and sports trainees. The app assesses each athlete's mental state and provides personalized daily activities to improve focus, emotional balance, and stress management—helping them perform better both on and off the field.

---

**🎯 THE CHALLENGE**

| Problem | Impact |
|---------|--------|
| Young athletes face immense pressure | Anxiety, burnout, performance drops |
| Mental health stigma in sports | Athletes avoid seeking help |
| Generic wellness apps | Not tailored for athletic performance |
| No progress tracking | Hard to measure mental improvement |
| Coaches lack tools | Can't monitor athlete mental wellness |

**User Pain Points:**
- **Athletes:** Stress before competitions, lack of coping tools
- **Coaches:** No visibility into athletes' mental state
- **Parents:** Worried about child's pressure handling

---

**👤 MY ROLE**

Complete ownership:
- Research on sports psychology techniques
- Assessment system design
- UI/UX design for the mobile app
- React Native development

## 2026 DESIGN TRENDS RESEARCH

### Sources Consulted

- [Muzli - Web Design Trends 2026](https://muz.li/blog/web-design-trends-2026/)
- [Designlab - UX/UI Portfolio Examples](https://designlab.com/blog/10-ux-ui-design-portfolios)
- [Index.dev - UI/UX Design Trends](https://www.index.dev/blog/ui-ux-design-trends)
- [Figma - Web Design Trends](https://www.figma.com/resource-library/web-design-trends/)
- [Promodo - UX/UI Design Trends](https://www.promodo.com/blog/key-ux-ui-design-trends)

### Key Trends to Implement

#### 1. Bento Grid Layouts
Multi-banner screens using modular grid layouts. Arranges content in blocks of different sizes creating visual hierarchy and dynamic rhythm.

```
┌──────────────────┬─────────────┐
│                  │             │
│    LARGE CARD    │  SMALL 1    │
│                  ├─────────────┤
│                  │  SMALL 2    │
├────────┬─────────┴─────────────┤
│ MEDIUM │       MEDIUM          │
└────────┴───────────────────────┘
```

#### 2. Kinetic Typography
Variable fonts and animated text that:
- Shift in weight on hover/scroll
- React to cursor movement
- Animate letter-by-letter on load
- Use gradient fills

#### 3. Purposeful Motion Design
Motion as communication tool:
- Button feedback animations
- Scroll-triggered reveals
- Page transitions
- Hover state changes
- Loading states

#### 4. Micro-Interactions (Micro Delight)
Subtle animations that make UI feel alive:
- Button press effects
- Toggle switches with physics
- Form field reactions
- Success/error feedback

#### 5. Bold Color Palettes
Moving away from muted tones:
- Neon gradients
- High-contrast pairings
- Y2K and retro influences
- "Dopamine design" aesthetics

#### 6. Glassmorphism & Depth
Layered visual effects:
- Frosted glass backgrounds
- Backdrop blur effects
- Gradient borders
- Shadow depth

#### 7. Interactive 3D Elements
WebGL-powered experiences:
- Cursor-reactive elements
- Scroll-triggered 3D
- Floating elements with parallax
- Interactive backgrounds

#### 8. Immersive Scroll Experiences
Beyond basic scroll animations:
- Horizontal scroll sections
- Parallax depth effects
- Snap scroll behavior
- Progress indicators
